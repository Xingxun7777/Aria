三、功能介绍（2:30 - 4:30）
————————————————————

【以下每个功能配合实际操作画面，画面承担演示，声音只讲感受和场景】

Aria 最基础的用法很简单。//
你按一下反引号键，/ 它就开始听。/
你说话，/ 它就直接把文字打到你光标那个位置。//
不用一直按着什么键，/ 就正常说话就行。/
说完了 / 它自己断句、自己输出。/ 再按一下就关了。↘

但如果只是语音转文字的话，/ 那跟别的工具也没什么区别了嘛。↗ //
Aria 真正不一样的地方 / 我觉得有这么几个。

【屏幕感知 — 画面：切到一个角色wiki页面，说出角色名，Aria正确识别】

比如说你切到一个网页，/ 上面写着"丰川祥子"。//
你一开口提到这个名字，/ 它就**直接认对了**。↘
不会给你打成什么"风川祥子"。//
因为它会自动去读你当前屏幕上的内容，/ 拿来当参考。//
这个你不用设置什么，/ 切过去说第一句话就生效了。↘

【热词纠错 — 画面：展示hotwords设置界面 + 对比演示】

然后就是专业术语的问题嘛。//
像 ComfyUI、Claude Code、DeepSeek 这种词，//
你跟普通语音输入说 / 它基本上没一个能认对的。↘

Aria 里面你可以自己加一个词表进去，//
告诉它这些词怎么写。//
下次你再说，/ 它就不会识别错了。//
而且就算它一开始听岔了，//
比如你说"迭代" / 它听成了"接待"，//
因为知道这俩读音很像，/ 它也会**自动帮你纠过来**。↘

【智能润色 — 画面：一段带口头禅的语音 → Aria输出干净文字】

还有一个我自己特别喜欢的功能。//
你说话的时候肯定会带一些 / "嗯"、"那个"、"就是说"这种口头禅嘛。↗ /
Aria 会自动帮你去掉。//
说了一大段话，/ 它也会自动帮你分段。/
出来的文字是**干净的**。↘

而且它会感知你当前在用什么软件。//
你在微信里说话，/ 它知道是聊天，/ 润色的时候保留口语感。↘
你在 Word 里说话，/ 它就偏书面一点。//
不用手动切，/ 它自己判断。↘

【语音指令 — 画面：前面开头已经演示过，这里快速回顾】

语音指令的话 / 开头已经演示过了嘛。//

五、视频简介文案
——————————————

Aria v1.0.2 — 免费开源的 Windows 本地 AI 语音输入工具

所有语音数据在本地处理，不上传任何服务器。

核心功能：
- 双引擎：Qwen3-ASR（多语言）/ FunASR（极速中文）
- 屏幕感知：自动读取当前窗口内容，识别更准
- 智能润色：口语过滤、自动分段、场景自适应
- 语音指令：翻译/回复/总结/定时提醒/打开文件
- 四层热词纠错 + 历史记录 + 完全离线
