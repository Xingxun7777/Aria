二、视频简介文案

Aria v1.0.2 — 免费开源的 Windows 本地 AI 语音输入工具。

所有语音数据在本地处理，不上传任何服务器。按一次快捷键开启，持续监听并自动识别，再按一次关闭。

核心功能：
- 双引擎：Qwen3-ASR（多语言）/ FunASR（极速中文），GPU 自动加速
- 四层热词纠错：ASR 引导 → 正则替换 → 拼音模糊 → AI 润色
- 屏幕感知：自动读取当前窗口内容，提升专业术语识别准确率
- 智能润色：口语过滤、自动分段、个性化规则、屏幕场景自适应
- 语音指令：唤醒词 + 命令，翻译/润色/回复/打开文件/定时提醒
- 历史记录：所有输入自动存储，按日期/类型/关键词检索

GitHub：https://github.com/Xingxun7777/Aria
下载：https://github.com/Xingxun7777/Aria/releases/tag/v1.0.2


三、口播文案（推荐版，约 3 分钟）

今天介绍一下我自己做的语音输入工具 Aria，最新版是 v1.0.2。

先说一下它是什么。简单来说，你按一下快捷键，Aria 就开始持续监听，你说话它就自动把文字打到光标那里。写文档、聊微信、写代码、发邮件，都能用。整个过程模型跑在你本地，不联网，不上传任何数据。

跟市面上的语音输入法比，Aria 有几个不一样的地方。

第一个是屏幕感知。你切到一个网页，比如你在看一个叫"丰川祥子"的角色页面，Aria 会自动读到你窗口标题里的这个名字。然后你说话的时候，它就知道你说的是"丰川祥子"而不是"风川祥子"。这个是即时生效的，切过去第一句话就能用。

第二个是四层热词纠错。你可以配一些专业术语，比如 ComfyUI、DeepSeek、Claude Code 这些，Aria 有四层管道来确保这些词能被正确识别。第一层是 ASR 上下文引导，第二层是正则替换，第三层是拼音模糊匹配，第四层是 AI 润色。每个词还可以设权重，精细控制。

第三个是智能润色。Aria 不只是帮你把语音转成文字，它还会帮你把文字理清楚。口语里的"嗯"、"那个"、"就是说"这些填充词，它会自动去掉。说了一大段话，它会自动帮你分段。你还可以自己写规则，比如"英文专有名词保留大小写"。而且它会根据你当前用的软件自动调整风格——在微信里保留口语感，在 Word 里偏书面化。

第四个是语音指令。你设一个唤醒词，比如"小助手"，然后就可以用语音来操作。选中一段文字说"小助手翻译一下"，它就弹窗翻译。说"小助手帮我回复"，它就生成回复建议。还可以说"小助手提醒我五分钟后喝水"。润色、总结、打开文件、撤销、复制粘贴，这些都支持。

最后说一下识别引擎。默认用的是 Qwen3-ASR，支持 52 种语言，VRAM 够的话自动用大模型，不够就用小的，没 GPU 也能跑 CPU 模式。启动的时候有 GPU 预热，所以第一句话也不会掉链子。

Aria 是免费开源的，代码在 GitHub 上。Lite 版不带模型，首次使用自动下载；Full 版带模型，解压即用。链接在简介里。

五、对比 Typeless 的话术要点

- Aria 完全免费开源 vs Typeless 每月 12 美元
- Aria 本地运行不上传数据 vs Typeless 云端处理
- Aria 有热词自定义系统 vs Typeless 无法自定义术语
- Aria 有屏幕感知（自动读窗口标题）vs Typeless 没有
- Aria 有语音指令系统（翻译/回复/打开文件）vs Typeless 只做输入
- 识别准确率和速度：两者接近，Aria 略有优势在中文专业术语场景
