# Changelog

All notable changes to Aria will be documented in this file.
Format based on [Keep a Changelog](https://keepachangelog.com/).

## [Unreleased]

### 新功能
- **三层混合 OCR 引擎** — 浏览器走 UI Automation（100% 精度读取 DOM），其他应用走 RapidOCR（PaddleOCR ONNX），兜底 WinRT OCR
- **OCR 关键词提取** — 从屏幕文字中提取 20 个关键词注入 ASR，过滤 UI 噪声，同音字识别准确率大幅提升
- **领域上下文自动注入润色层** — 使用场景设置自动影响 AI 润色判断（如 3D 建模场景纠正"景点"为"顶点"）
- **路径打开增强** — 支持缺盘符、缺反斜杠、相对路径、用户子目录等不完整路径的智能补全
- **终端自动切换输入模式** — Typewriter 模式检测到终端应用时自动切换为剪贴板模式，解决 Windows Terminal 输入失效
- **贴纸弹出动画** — 350ms 渐入 + 上滑弹出（OutBack 缓动）

### 改进
- **右键菜单重写** — 3 种润色模式（关闭/本地/高质量），翻译输出改为 toggle 开关，设置和历史并排按钮
- **设置面板 tooltip** — 所有交互控件添加悬停提示
- **噪声过滤优化** — 短句阈值从 3 字降为 1 字，不再误杀 2-3 字的正常短句
- **润色 prompt 脱敏** — 移除个人相关示例，适合分发

### 修复
- **Whisper 幻觉词表移除** — "字幕"等词不再被误判为幻觉（Qwen3/FunASR 不产生此类幻觉）
- **中文重复字不再误杀** — "落落落落伍"等 ASR 重复输出不再被 Pattern 4 当幻觉丢弃
- **上下文泄漏检测修复** — 移除 5 秒限制，改为比率检测（12 字/秒）适用所有音频长度
- **Stale OCR 防护** — OCR 结果绑定窗口 + 时间戳，切窗口或超 10 秒自动丢弃旧 context
- **ONNX Runtime DLL 路径修复** — 解决与 PyTorch CUDA 12.8 的 DLL 搜索冲突

## [1.0.2] - 2026-03-19

### 新功能
- **屏幕 OCR 上下文** — VAD 检测到语音时自动 OCR 当前窗口文字，注入 ASR 上下文提升识别准确率
- **历史记录系统** — JSONL 按日分片存储，支持搜索/过滤/导出/删除，90 天自动清理
- **AI 回复弹窗** — 选中文字说"帮我回复"，AI 生成回复并弹窗展示
- **噪声过滤** — ASR 后自动过滤无意义填充词（嗯/啊/哦/呃 等 16 种）
- **ASR 近期上下文** — 最近 10 条识别结果作为滚动上下文，提升连续对话准确率
- **上下文泄漏检测** — 检测 ASR 输出是否为上下文回声（时长/子串双重检查）
- **语音开路径** — 选中文本说"帮我打开"，智能识别并打开文件/文件夹/URL
- **语音提醒/闹钟** — 语音设置定时提醒，支持自然语言时间解析
- **语音指令指南** — 设置面板新增指令速查弹窗
- **贴纸彩蛋** — 命令执行成功时显示动画贴纸

### 改进
- **润色提示词优化** — 分场景风格引导（聊天/文档/编程），改进口语过滤和自动结构化
- **API 预设扩展** — 16 个 API 提供商预设 + URL 自动识别 + 模型列表拉取
- **翻译弹窗** — 支持拖动/置顶/插入按钮，暖色调重设计
- **浮动球** — Pipeline 慢阶段辉光动画提示
- **高亮/历史按钮** — 修复在 pythonw.exe 下不工作的问题

### 修复
- reply_style 配置因相对路径在非项目 CWD 下加载失败
- LocalPolishEngine 缺少 screen_context 参数导致输出完全失败
- Typewriter 模式换行符在终端中触发 Enter（关键修复）
- ASR max_new_tokens 256→1024 修复长句被截断
- 短音频过滤 + 跳过微量文本润色，减少噪声浪费
- 上下文泄漏检测、OCR 文本清洗、缓冲区去重
- 多个 UnboundLocalError / NameError 崩溃修复

## [1.0.0] - 2026-02-28

首个公开发布版本。

### 语音识别
- **双引擎架构** — Qwen3-ASR（默认，多语言 + 上下文热词）和 FunASR（Paraformer-zh，轻量备选）
- **Silero-VAD 语音检测** — 自动区分语音与静默，带能量门控 fallback
- **三级防幻觉系统** — Pre-ASR 能量门控 + 三档能量泄漏检测 + retry-without-context
- **CUDA GPU 加速** — 自动检测显卡，不兼容时平滑回退 CPU

### 热词纠错
- **四层纠错管线** — initial_prompt → 正则替换 → 拼音模糊匹配 → AI 润色
- **热词权重系统** — 每个热词可独立设置 0-1.0 权重，精细控制各层参与度
- **AI 润色层** — 支持 OpenRouter 等 API，可选开启

### 语音控制
- **唤醒词指令** — 语音键盘命令（发送、删除、撤销、复制等）
- **模式切换** — 开启/关闭自动发送、休眠/唤醒
- **Typewriter 模式** — 逐字符输入，兼容游戏和管理员权限应用

### 选区功能
- **选区指令** — 润色、翻译、扩写、缩写选中文本
- **翻译弹窗** — 选中文字说"什么意思"，浮窗显示翻译
- **AI 对话窗口** — 选中文字说"问AI"，发起上下文对话

### 界面
- **浮动球** — 可拖动状态指示器，左键切换录音，右键菜单
- **流式字幕面板** — 毛玻璃半透明面板实时显示识别中间结果
- **系统托盘** — 后台运行，右键快捷操作
- **设置面板** — 图形化全功能配置（引擎、热词、快捷键、音频设备等）
- **启动画面** — 加载进度展示

### 系统集成
- **全局热键** — 默认 `` ` `` 键切换录音
- **开机自启动** — Registry HKCU\Run 方式，支持路径变更自修复
- **管理员提权检测** — 目标应用需高权限时自动提示
- **单例检查** — Windows Named Mutex + 文件锁 + PID 创建时间校验

### 构建与分发
- **便携版打包** — 嵌入式 Python，开箱即用，无需安装
- **桌面快捷方式生成器** — 一键创建快捷方式
- **配置热重载** — 修改配置文件后自动生效，无需重启

### 技术架构
- QtBridge 线程安全信号机制
- 原子性 JSON 写入（tmp + fsync + os.replace）
- ASR 队列有界（maxsize=5）+ 丢弃策略
- 配置五层 fallback 链（widget → config → app → template → dataclass）
