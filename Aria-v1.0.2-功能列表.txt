Aria v1.0.2 完整功能列表
========================

Windows 本地 AI 语音输入工具
所有语音数据在本地处理，不上传任何服务器。


一、语音识别
──────────

  • 双引擎架构 — Qwen3-ASR（默认，多语言 + 上下文热词）/ FunASR（中文，极速）
  • CUDA GPU 自动加速 — 检测显卡，不兼容时自动回退 CPU 模式
  • Silero-VAD 语音检测 — 区分语音与静默，带能量门控 fallback
  • 三级防幻觉系统 — Pre-ASR 能量门控 + 能量泄漏检测 + retry-without-context
  • 实时流式字幕 — 说话时毛玻璃面板显示中间识别结果
  • 噪声过滤 — 自动丢弃无意义填充词（嗯/啊/哦/呃/嘶 等 19 种模式）
  • 屏幕感知 — 自动感知当前窗口内容，提升专业术语和人名识别准确率
    - 窗口标题即时提取（0ms，切窗口后第一句话就生效）
    - 后台 OCR 补充页面详细内容
    - 切窗口/切标签自动刷新，结果按窗口缓存
  • RMS 音量自适应 — 自动增益小音量语音（最大 5 倍），改善低声说话的识别准确率
  • 近期上下文 — 最近 10 条识别结果作为滚动上下文，提升连续对话准确率
  • 上下文泄漏检测 — 检测 ASR 输出是否为上下文回声（时长/子串双重检查）


二、热词纠错
──────────

  • 四层纠错管线 — ASR 引导 → 正则替换 → 拼音模糊匹配 → AI 润色
  • 热词权重系统 — 每个热词可设 0-1.0 权重，精细控制各层参与度
    - 0：禁用
    - 0.1：谨慎（仅正则 + AI 润色严格模式）
    - 0.3：提示（默认，正则 + AI 润色 hint）
    - 0.5：参考（加入 ASR 上下文偏置）
    - 1.0：关键（全部层级生效 + 拼音模糊匹配）
  • AI 润色层 — 支持 OpenRouter / DeepSeek 等 API


三、智能润色
──────────

  • 高质量模式 — 通过 API 调用大语言模型修正错别字、标点和同音字
  • 口语过滤 — 自动去除"就是"、"然后的话"等口语填充词
  • 自动结构化 — 长段口述自动整理为带换行、编号的结构化文本
  • 个性化规则 — 自然语言描述润色偏好（如"英文专有名词保留原始大小写"）
  • 屏幕感知 — 聊天场景保留口语感，文档场景偏书面化，编程场景保留标识符
  • 本地润色 — 可自行配置 GGUF 模型实现离线润色
  • 备用 API 智能轮询 — 主/备 API，连续 2 次慢响应自动切换


四、选区指令
──────────

  选中文字后，说唤醒词 + 命令：

  • 润色 / 优化 — 提升文字表达质量
  • 翻译成英文 / 中文 / 日文 — 替换为对应语言翻译
  • 扩写 / 展开 — 增加细节和深度
  • 缩写 / 精简 — 保留核心信息
  • 重写 / 改写 — 不同表达方式
  • 什么意思 — 弹窗翻译（不替换原文）
  • 总结一下 — 弹窗显示摘要
  • 帮我回复 — AI 生成回复建议，支持指定风格（如"帮我回复，语气轻松一点"）
  • 问 AI — 打开 AI 对话窗口
  • 记一下 + 内容 — 记录想法
  • 帮我打开 — 智能识别并打开文件/文件夹/URL


五、弹窗交互
──────────

  翻译、总结、回复弹窗支持：
  • 拖拽移动
  • 固定（Pin）防止自动消失
  • 一键复制
  • 一键插入（自动切回原窗口粘贴）


六、唤醒词控制
────────────

  • [唤醒词] 开启/关闭自动发送 — 控制识别后是否自动按回车
  • [唤醒词] 休眠 / 醒来 — 暂停/恢复语音监听
  • [唤醒词] 记一下 + 内容 — 语音记录想法
  • 唤醒词可自定义，内置可选：遥遥、瑶瑶、小朋友、小溪、助手、小白


八、语音提醒/闹钟
───────────────

  • 说 [唤醒词] 提醒我 + 时间 + 内容
  • 支持自然语言时间解析（如"5分钟后"、"下午3点"）
  • 到时间后弹窗提醒


九、历史记录
──────────

  • 所有语音输入、翻译、润色、回复结果自动存储
  • 历史浏览器 — 按日期翻页、按类型筛选、搜索关键词
  • 支持导出为 Markdown
  • 自动清理过期记录（默认 90 天）


十、界面
──────

  • 浮动球 — 可拖动状态指示器
    - 左键：切换录音
    - 右键：弹出菜单（润色模式/翻译/字幕/锁定/休眠/设置/历史）
    - 中键：锁定位置
    - 状态色：蓝色待机 / 绿色录音中 / 橙色处理中 / 灰色休眠
  • 右键菜单功能
    - 润色模式切换：关闭 / 本地润色 / 高质量（三选一）
    - 翻译弹窗开关（开=弹窗，关=复制到剪贴板）
    - 实时字幕开关
    - 锁定位置开关
    - 休眠模式开关
    - 设置 / 历史快捷按钮
    - 启用/禁用总开关
  • 流式字幕面板 — 毛玻璃半透明实时显示
  • 系统托盘
    - 单击：打开历史记录
    - 双击：打开唤醒词设置
    - 右键：快捷菜单
  • 设置面板 — 图形化全功能配置，30+ 悬浮提示（Tooltip）
  • AI 对话窗口 — 深色主题聊天界面，支持流式输出
  • 启动画面 — 加载进度展示
  • 浮动球辉光 — Pipeline 慢阶段动画提示
  • 贴纸彩蛋 — 命令执行成功时显示动画贴纸（350ms 渐入 + 弹出动画）
  • 语音指令指南 — 设置面板速查弹窗


十一、系统集成
────────────

  • 全局热键 — 默认 ` 键（反引号）切换录音，可自定义
  • 开机自启动 — Registry HKCU\Run 方式，支持路径变更自修复
  • 管理员提权检测 — 目标应用需高权限时自动提示
  • 单例检查 — Named Mutex + 文件锁 + PID 创建时间校验
  • Typewriter 模式 — 逐字输入，兼容游戏和管理员权限应用
  • 终端自动检测 — 终端窗口自动切换剪贴板模式（cmd/PowerShell/Terminal 不支持逐字输入）
  • 智能路径打开 — 识别选中文本中的文件路径/URL，支持残缺路径补全
  • 配置热重载 — 修改配置后自动生效，无需重启
  • 翻译输出切换 — 弹窗显示 / 复制到剪贴板
  • 回复风格 — 自定义 AI 回复的风格偏好


十二、构建与分发
──────────────

  • 便携版打包 — 嵌入式 Python，解压即用，无需安装
  • Lite 版 — 不含模型，首次启动自动下载（GitHub 发布）
  • Full 版 — 内置 Qwen3-ASR 模型，解压即用
  • 桌面快捷方式生成器 — 一键创建快捷方式
  • HF 镜像加速 — 中国用户默认使用 hf-mirror.com


十三、系统要求
────────────

  操作系统：Windows 10/11 64 位
  Python：3.12（便携版已内置）
  内存：最低 8 GB，推荐 16 GB
  GPU：可选，NVIDIA GTX 16xx+（4 GB VRAM），无 GPU 自动 CPU 模式

  GPU 兼容性：
  • RTX 40xx / 50xx（Ada / Blackwell）— 完全支持
  • RTX 30xx（Ampere）— 完全支持
  • RTX 20xx / GTX 16xx（Turing）— 支持
  • GTX 10xx（Pascal）— 自动 CPU 回退
  • 无 NVIDIA GPU — CPU 模式
